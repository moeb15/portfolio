Outfit Segmentation With UNet

Segmenting outfits using UNet

 Posted on May 21, 2023  |  4 minutes  |  799 words  |  Mujtaba Bader

Project Overview

For this project I took a dataset from Kaggle consisting of images of a person wearing some outfit, and a corresponding mask of the outfit.

Creating the dataset

I started by getting the directories of the images and masks then converting the images into numpy arrays using OpenCV,

img_dataset = []
mask_dataset = []

for idx in range(len(imgs)):
    img = cv2.imread(f'{IMG_DIR}{imgs[idx]}',cv2.IMREAD_UNCHANGED)
    mask = cv2.imread(f'{MASKS_DIR}{masks[idx]}',cv2.IMREAD_UNCHANGED)
    img_resized = cv2.resize(img,(128,128))
    mask_resized = cv2.resize(mask,(128,128))

    img_dataset.append(img_resized)
    mask_dataset.append(mask_resized)

img_dataset = np.array(img_dataset)
mask_dataset = np.array(mask_dataset)

After the I had the data loaded in, I then normalized the data and created my train and validation datasets

dataset = tf.data.Dataset.from_tensor_slices((img_dataset,mask_dataset))

BATCH_SIZE = 64
BUFFER_SIZE = 10000

Creating the model

The UNet model consists of an encoder and a decoder, you can think of the encoder as a model that takes our image then crunches it down to some fixed size vector representation of our image, kind of like a word embedding, and our decoder model as performing the opposite, reconstructing our image using the output of the encoder. To help with building the encoder and decoder I defined two helper functions, one for the downsampling blocks in the encoder, and another for the upsampling blocks in the decoder,

def downsample_block(prev_block,filters,kernel_size,dropout):
    x = tf.keras.layers.Conv2D(filters=filters,
                               kernel_size=kernel_size,
                               activation='relu',
                               padding='same')(prev_block)
    if dropout != None:
        x = tf.keras.layers.Dropout(dropout)(x)
    x = tf.keras.layers.Conv2D(filters=filters,
                               kernel_size=kernel_size,
                               activation='relu',
                               padding='same')(x)
    return x

def upsample_block(prev_block,skip_conn,filters,
                   kernel_size,dropout,strides):
    x = tf.keras.layers.Conv2DTranspose(filters=filters,
                                        kernel_size=kernel_size[0],
                                        activation='relu',
                                        strides=strides,
                                        padding='same')(prev_block)
    x = tf.keras.layers.Concatenate(axis=3)([x,skip_conn])
    x = tf.keras.layers.Conv2D(filters=filters,
                               kernel_size=kernel_size[1],
                               activation='relu',
                               padding='same')(x)
    if dropout != None:
        x = tf.keras.layers.Dropout(dropout)(x)
    x = tf.keras.layers.Conv2D(filters=filters,
                               kernel_size=kernel_size[1],
                               activation='relu',
                               padding='same')(x)
    return x

After writing the helpers I defined the model the full model,

def UNet(img_shape):

    filters = 32
    enc_kernel_size = 3
    dec_kernel_size = [2,3]

    img_in = tf.keras.layers.Input(shape=img_shape)

    # Encoder
    d1 = downsample_block(img_in,filters,enc_kernel_size,0.1)
    pool1 = tf.keras.layers.MaxPooling2D((2,2))(d1)
    
    d2 = downsample_block(pool1,2*filters,enc_kernel_size,0.1)
    pool2 = tf.keras.layers.MaxPooling2D((2,2))(d2)

    d3 = downsample_block(pool2,4*filters,enc_kernel_size,0.1)
    pool3 = tf.keras.layers.MaxPooling2D((2,2))(d3)

    d4 = downsample_block(pool3,8*filters,enc_kernel_size,0.1)
    pool4 = tf.keras.layers.MaxPooling2D((2,2))(d4)
    
    d5 = downsample_block(pool4,16*filters,enc_kernel_size,0.3)

    # Decoder
    u1 = upsample_block(d5,d4,8*filters,dec_kernel_size,0.2,2)
    u2 = upsample_block(u1,d3,4*filters,dec_kernel_size,0.2,2)
    u3 = upsample_block(u2,d2,2*filters,dec_kernel_size,0.2,2)
    u4 = upsample_block(u3,d1,filters,dec_kernel_size,0.2,2)

    out_layer = tf.keras.layers.Conv2D(filters=1,kernel_size=1,activation='sigmoid')(u4)

    return tf.keras.Model(inputs=img_in,outputs=out_layer,name='U-Net')

Once this had been done I moved onto training the model.

First iteration

After training the model and analyzing the results, I noticed that it had been failing to improve past 76% accuracy and had failed at reducing the loss past a certain point,

As you can see, the model clearly isn’t overfitting the training data, the issue is the opposite, the model is high bias, it fails to capture the important features of our dataset. The mask produced by the model isn’t terrible, but it’s clear that the model is failing to capture the detail of the true mask, let’s try and improve the model.

Some things we can consider is increasing the number of filters in our CNN layers and increasing the amount of layers in our network, we could also use data augmentation to increase the size of our dataset, keras has layers we could use directly in our model to handle this. Another option is to use a different model entirely and see how it performs in comparison.

Second Iteration

After playing around a little bit I tweaked some hyperparameters like the learning rate, increased the the number of layers, filters, and added data augmentation, and decreased the amount of data in the validation set to increase the amount of data in the in the training set, seeing as the dataset only has 1000 images. The following code shows the changes to the mode

def UNet(img_shape):

    filters = 48
    enc_kernel_size = 3
    dec_kernel_size = [2,3]

    img_in = tf.keras.layers.Input(shape=img_shape)
    
    # Data Augmentation
    aug_flip = tf.keras.layers.RandomFlip("horizontal_and_vertical")(img_in)
    aug_rotate = tf.keras.layers.RandomRotation(0.2)(aug_flip)

    # Encoder
    d1 = downsample_block(aug_rotate,filters,enc_kernel_size,0.1)
    pool1 = tf.keras.layers.MaxPooling2D((2,2))(d1)
    
    d2 = downsample_block(pool1,2*filters,enc_kernel_size,0.1)
    pool2 = tf.keras.layers.MaxPooling2D((2,2))(d2)

    d3 = downsample_block(pool2,4*filters,enc_kernel_size,0.1)
    pool3 = tf.keras.layers.MaxPooling2D((2,2))(d3)

    d4 = downsample_block(pool3,8*filters,enc_kernel_size,0.1)
    pool4 = tf.keras.layers.MaxPooling2D((2,2))(d4)
    
    d5 = downsample_block(pool4,16*filters,enc_kernel_size,0.1)
    pool5 = tf.keras.layers.MaxPooling2D((2,2))(d5)
    
    d6 = downsample_block(pool5,32*filters,enc_kernel_size,0.3)

    # Decoder
    u1 = upsample_block(d6,d5,16*filters,dec_kernel_size,0.2,2)
    u2 = upsample_block(u1,d4,8*filters,dec_kernel_size,0.2,2)
    u3 = upsample_block(u2,d3,4*filters,dec_kernel_size,0.2,2)
    u4 = upsample_block(u3,d2,2*filters,dec_kernel_size,0.2,2)
    u5 = upsample_block(u4,d1,filters,dec_kernel_size,0.2,2)

    out_layer = tf.keras.layers.Conv2D(filters=1,kernel_size=1,activation='sigmoid')(u5)

    return tf.keras.Model(inputs=img_in,outputs=out_layer,name='U-Net')

The changes overall didn’t improve perfomance a signficiant amount (from 76.47% to 76.67% accuracy on the validation set), however I did kind of like the images produced by the model, that being said it may be good to consider a different model for the problem.
