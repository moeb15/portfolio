Sentiment Analysis of Tweets Dataset Obtained From Kaggle

 Posted on April 22, 2023  |  1 minutes  |  182 words  |  Mujtaba Bader

Project Overview

In this project I cleaned, transformed, and analyzed a dataset of 4893 tweets, each of which had a label attached indicating the sentiment of the tweet. Using GloVe word embeddings and Tensorflow I created a model that learned to perform sentiment analysis of tweets based off the given dataset.

Steps Followed

The following are the steps I took throughout the project,

  • Clean and transform data, removing non-alpahnumeric characters from tweets and label encoding sentiment labels
  • Tokenize and pad tweets, build embedding matrix using weights obtained from GloVe model
  • Build model using embedding layer, 1d convolutional layers, pooling layers, and dense layer for classification

For a deeper look at how this was done you can take a look at the github repository to get a more detailed look at the project.
